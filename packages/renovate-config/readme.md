# renovate-config-fisker

> fisker's shareable renovate configuration

## Usage

```sh
echo {"extends":["github>fisker/shared-configs//packages/renovate-config/default"]}> .github/renovate.json5
```

```sh
echo {"extends":["@fisker"]}> renovate.json
```
