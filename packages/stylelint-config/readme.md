# @fisker/stylelint-config

> fisker's Shareable stylelint Configuration

## Install

```sh
yarn add --dev stylelint prettier @fisker/stylelint-config
```

## stylelint.config.js

```js
module.exports = {
  extends: ['@fisker/stylelint-config'],
}
```
