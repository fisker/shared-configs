# @fisker/commitlint-config

> fisker's Shareable commitlint Configuration

## Install

```sh
yarn add prettier @commitlint/cli @fisker/commitlint-config --dev
```

## Usage

```js
module.exports = {extends: ['@fisker/commitlint-config']}
```
