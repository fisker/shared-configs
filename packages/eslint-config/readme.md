# @fisker/eslint-config

> fisker's Shareable ESLint Configuration

## Install

```sh
yarn add --dev eslint prettier @fisker/eslint-config
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker'],
}
```
