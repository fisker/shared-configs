# @fisker/eslint-config-ava

> fisker's Shareable ESLint Configuration for ava test

## Install

```sh
yarn add --dev @fisker/eslint-config @fisker/eslint-config-ava
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker', '@fisker/ava'],
}
```
