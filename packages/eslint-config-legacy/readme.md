# @fisker/eslint-config-legacy

> fisker's Shareable ESLint Configuration for legacy project

## Install

```sh
yarn add --dev eslint prettier @fisker/eslint-config-legacy
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker/legacy'],
}
```
