# @fisker/eslint-config-jest

> fisker's Shareable ESLint Configuration for jest test

## Install

```sh
yarn add --dev @fisker/eslint-config @fisker/eslint-config-jest
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker', '@fisker/jest'],
}
```
