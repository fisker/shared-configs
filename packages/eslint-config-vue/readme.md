# @fisker/eslint-config-vue

> fisker's Shareable ESLint Configuration for vue project

## Install

```sh
yarn add --dev eslint prettier @fisker/eslint-config-vue
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker/vue'],
}
```
