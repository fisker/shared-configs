# @fisker/eslint-config-mocha

> fisker's Shareable ESLint Configuration for mocha test

## Install

```sh
yarn add --dev @fisker/eslint-config @fisker/eslint-config-mocha
```

## .eslintrc.js

```js
module.exports = {
  extends: ['@fisker', '@fisker/mocha'],
}
```
